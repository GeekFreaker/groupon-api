Reseller API
============

Пользователи
------------

### Подробная информация о текущем пользователе

#### GET /resellers/user.json

Возвращает подробную информацию о текущем пользователе.

*Ответ сервера:*

```json
{
  "user": {
    "email": "atipugin@darberry.ru",
    "id": 27256253,
    "balance": 2530.0
  }
}
```
